# Architecture Decision Records

This directory records significant architectural decisions for this project, one
file per decision. Each ADR captures the context, the decision, and its
consequences so the reasoning survives beyond the original conversation.

Format: a lightweight [Nygard-style](https://cognitect.com/blog/2011/11/15/documenting-architecture-decisions)
ADR. Number files sequentially (`NNNN-title.md`) and never edit an accepted
record in place — supersede it with a new ADR and update its status.
